# Martani 클라우드 AI 플랫폼 가이드

## 플랫폼 소개
Martani(마타니)는 클라우드 스토리지, AI 비서, 데이터 마이닝, 문서 분석을 통합한 AI 클라우드 플랫폼입니다.
RAG(검색 증강 생성)와 시맨틱 검색으로 문서 내용을 이해하고, AI 비서가 파일 관리부터 웹 자동화까지 다양한 작업을 수행합니다.

---

## 사이드바 메뉴 구조

```
클라우드
├── 탐색기 (/files)      - 파일 업로드/관리/공유
├── 인덱싱 (/indexing)   - 문서 벡터화 + 시맨틱 검색
└── 보안관 (/vault)      - 자격증명/파일 암호화 금고

인공지능
├── 메신저 (/chat)       - AI 비서 대화
├── 마이닝 (/ai/data/mining) - 워크플로우 기반 데이터 수집/정제/전달
└── 스케줄 (/ai/schedule)    - 워크플로우 반복 실행 설정
```

---

## 주요 기능 상세

### 1. 클라우드 탐색기 (/files)
파일을 업로드/다운로드/공유/관리하는 핵심 기능.

**파일 작업:**
- 업로드: 드래그&드롭 또는 버튼으로 다중 파일 업로드
- 다운로드: 컨텍스트 메뉴에서 다운로드
- 공유: 만료일/비밀번호 설정 가능한 공유 링크 생성 (7일 기본)
- 이름 변경, 이동(폴더 선택 모달), 삭제
- 일괄 선택 → 일괄 삭제/이동

**폴더 관리:**
- 폴더 생성, 브레드크럼 경로 탐색
- 리스트 뷰 / 그리드 뷰 전환

**인덱싱 상태 표시:**
- 완료(녹색), 처리중(파랑+진행률), 실패(빨강+재시도), 미인덱싱(회색)

**스토리지:**
- 사용량/할당량 진행바 표시, 초과 시 업로드 차단
- MinIO 기반 S3 호환 스토리지

**지원 파일:**
- 문서: PDF, DOCX, TXT, MD, XLSX, XLS, CSV
- 이미지: PNG, JPG, JPEG, GIF, WEBP
- 오디오: MP3, WAV, OGG, FLAC, M4A
- 동영상: MP4, AVI, MKV, WEBM

---

### 2. AI 메신저 (/chat)
AI 비서(마타니)와 대화하는 채팅 인터페이스.

**빈 상태(세션 없을 때):**
- "마티니와 대화를 시작하세요" 안내 화면
- [새 대화] 버튼 → 카테고리 선택 모달 (전체 또는 특정 카테고리)
- [대화 목록] 버튼 → 기존 대화 불러오기/삭제

**카테고리 기반 RAG:**
- 카테고리 없음(전체): 모든 인덱싱된 파일에서 검색
- 특정 카테고리 선택: 해당 카테고리에 분류된 파일만으로 검색 범위 제한

**채팅 기능:**
- 스트리밍 응답 (실시간 타이핑)
- 세션별 대화 기록 유지
- 파일 첨부 → AI에게 분석 요청
- 마크다운 렌더링 (코드 하이라이팅, 표, 목록 등)
- 메시지 검색, 대화 내보내기(Markdown)

**AI 비서 도구 (24개):**
- 파일 관리(8): 목록조회, 정보확인, 이름검색, 내용검색, 텍스트파일생성, 파일삭제, 읽기, 일괄이동
- 메모(4): 목록, 읽기, 생성/수정, 검색
- 메일(3): 발송, 상태변경, 삭제
- 웹(2): 웹 검색(SearXNG), URL 콘텐츠 가져오기
- 유틸(1): 현재 시간 조회
- 메모리(2): 기억 저장, 기억 회상
- 브라우저(6): 웹 페이지 탐색, 읽기, 클릭, 입력, 스크롤, 스크린샷
- 스케줄(4): 예약 작업 목록/생성/수정/삭제
- 데이터 수집(4): 수집 작업 생성/목록/실행/결과조회

**도구 호출 시각화:**
- 도구 체인 표시 (어떤 도구를 사용 중인지 실시간 표시)
- 바운싱 도트 애니메이션 (대기 중)

### 3. 인덱싱 & 시맨틱 검색 (/indexing)
업로드된 파일을 벡터 임베딩으로 인덱싱하여 자연어 검색 가능.

**카테고리 관리:**
- 파일을 카테고리로 분류 (색상 8종: 파랑, 빨강, 초록, 노랑, 보라, 주황, 분홍, 회색)
- 카테고리별 파일 수 표시
- 메신저에서 카테고리별 대화 생성 시 검색 범위 제한에 활용

**인덱싱 통계:**
- 전체 파일 수, 완료, 처리중, 실패, 대기

**시맨틱 검색:**
- 자연어 질의 (키워드 아닌 의미 기반 검색)
- 카테고리 필터, 타입 필터 (전체/텍스트/이미지/오디오/비디오)
- 검색 결과: 파일명, 유사도(%), 매칭 텍스트 미리보기

**처리 지원:**
- 문서: PDF, DOCX, TXT, MD → 텍스트 추출 + 청킹 + 임베딩
- 스프레드시트: XLSX, XLS, CSV → 시트별 분석 + 임베딩
- 이미지: PNG, JPG 등 → Vision AI로 설명 생성 + 임베딩
- 오디오: MP3, WAV 등 → Whisper STT로 텍스트 변환 + 임베딩

---

### 4. 보안관 (/vault)
자격증명과 파일을 AES-256 암호화로 안전하게 보관하는 금고.

**자격증명 금고:**
- 웹사이트 로그인 정보 (사이트명, 아이디, 비밀번호, 메모) 암호화 저장
- 비밀번호 보기/숨기기 토글
- AI 비서 브라우저 자동 로그인 시 사용

**API 키 금고:**
- 외부 서비스 API 키 암호화 저장 (사이트명, API 키, 만료일, 메모)

**파일 금고:**
- 탐색기에서 파일 → "금고에 보관" → AES-256 암호화 이동
- "꺼내기" 버튼으로 복호화 후 탐색기 복원
- 금고 내 파일 영구 삭제 가능 (복구 불가)

---

### 5. 마이닝 (/ai/data/mining)
React Flow 기반 비주얼 워크플로우 에디터로 데이터 수집→정제→전달 파이프라인을 구성.

**작업자 관리:**
- [새 작업자] 버튼 → 이름 입력 다이얼로그 → 워크플로우 생성
- [작업자 목록] 버튼 → 저장된 파이프라인 목록 (검색/삭제)
- 빈 상태: 안내 화면 + "새 작업자 만들기" CTA

**워크플로우 캔버스:**
- 노드 3종을 드래그해서 배치:
  - 수집(Collect): URL 목록 + 크롤링 엔진 선택 + 프롬프트
  - 정제(Refine): 정제 프롬프트 + 출력 형식(JSON/CSV/요약) + 필터 규칙
  - 전달(Deliver): Webhook/Email/Cloud Folder 전달 설정
- 노드 간 엣지(선)로 데이터 흐름 연결
- 노드 클릭 → 우측 설정 패널

**크롤링 엔진:**
- Crawl4AI (기본, 빠름)
- Scrapling (범용)
- Stealth (봇 감지 회피)

**실행:**
- "워크플로우 실행" 버튼 → 수집→정제→전달 순차 실행
- 각 노드에 실시간 상태 표시 (대기/실행중/완료/에러)
- 엣지 애니메이션으로 진행 상황 시각화

---

### 6. 스케줄 (/ai/schedule)
마이닝 작업자(파이프라인)의 반복 실행을 설정.

**스케줄 설정:**
- 프리셋: 매 시간, 매 3시간, 매 6시간, 매일 오전 9시, 매일 오후 6시, 매주 월요일, 매월 1일
- 커스텀: Cron 표현식 직접 입력 (예: `0 */6 * * *`)
- 활성/비활성 토글
- 스케줄 해제

**작업자 목록:**
- 필터: 전체 / 스케줄 설정됨 / 미설정
- 각 작업자의 스케줄 상태, 연결된 수집/정제/전달 표시
- 일시정지/활성화, 스케줄 편집, 삭제

---

### 7. 관리자 기능 (/admin) — admin 권한 전용

**대시보드:**
- 전체/활성 사용자 수, 파일 수, 스토리지 사용량

**사용자 관리:**
- 사용자 CRUD (이메일, 이름, 역할, 활성 상태, 할당량)
- 플랜 관리: Basic/Pro, 토큰 할당량

**시스템 설정:**
- LLM API 설정 (URL, API 키, 모델 선택)
- 에이전트 시스템 프롬프트 커스터마이징
- 에이전트 도구 활성화/비활성화

**감사 로그:**
- 사용자 활동 기록 조회 (로그인, 파일 작업, 채팅 등)

## 메모 (/notes) — 사이드바 숨김, URL 직접 접근 가능
포스트잇 스타일 자유 배치 메모장.
- 7가지 색상, 드래그 이동, 크기 조절, 핀 고정
- 격자/세로/가로 정렬 도구
- Ctrl+N 새 메모, Ctrl+D 삭제
- AI 비서 도구로 메모 CRUD 가능

## 인증 & 계정

**로그인 (/login):**
- 이메일/비밀번호 인증, 이메일 인증 필수
- 로그인 실패 횟수 추적

**회원가입 (/register):**
- 이메일, 비밀번호(8자 이상), 이름(선택)
- 가입 후 이메일 인증 발송

**사용자 플랜:**
- Basic/Pro 플랜, 월별 AI 토큰 할당량, 매월 1일 자동 초기화

## 기술 정보

**AI 모델:**
- 텍스트: OpenRouter (openai/gpt-4o-mini)
- 비전: OpenRouter (google/gemini-2.5-flash)
- 임베딩: Ollama (bge-m3, 1024차원, CPU)
- 음성 전사: whisper.cpp (CPU)

**인프라:**
- 백엔드: FastAPI + SQLAlchemy + Celery
- 프론트: Next.js 14 + React 18 + TailwindCSS
- DB: PostgreSQL 16 + pgvector
- 스토리지: MinIO (S3 호환)
- 검색 엔진: SearXNG (자체 호스팅)
- 워크플로우: React Flow v12

**LLM 안정성:**
- OpenRouter API 500/502/503 에러 시 자동 재시도 (최대 3회, 1s→2s→4s 백오프)
- 에러 시 사용자에게 친절한 한국어 메시지 표시
